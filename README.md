# 将棋ノート (Shogi Note) - GitHub Storage Edition

将棋の棋譜や関連リンクを管理するための自分専用ツールです。
データは **自分自身の GitHub リポジトリ** に保存されます。

## 主な機能

- **棋譜管理**: ぴよ将棋などの棋譜テキストを貼り付けて保存。タグ付けや分析が可能。
- **リンク集**: 将棋関連の定跡記事や動画などをフォルダ形式で整理。
- **QRコードログイン**: 一度設定すれば、URL情報を使って自動ログインできます。

## セットアップ方法

1.  **リポジトリをインポート**:
    - このリポジトリを自分の GitHub アカウントに「Private」リポジトリとしてインポートしてください。
    - https://github.com/new/import

2.  **下記リンクへアクセス**:
    - https://github.com/MByte-Violin/Public-Shogi-note

3.  **アプリにアクセスしてログイン**:
    - 発行されたURLにアクセスします。
    - 右上の **設定（歯車アイコン）** を開き、以下の情報を入力して「設定を保存して接続」をクリックします。
        - **Owner**: あなたの GitHub ユーザー名。
        - **Repository Name**: あなたが命名したリポジトリ名。
        - **GitHub Token**: あなたが発行したGithubアクセストークン (repo 権限が必要)。

4.  **自動ログインの活用**:
    - 接続に成功すると、設定画面に QRコード や共有用URLが表示できるようになります。
    - このURL情報で、次回から設定不要で自動ログインできます。ただし他人に知られない様に注意してください
